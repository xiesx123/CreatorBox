Configuration Options ‚Äã

Model ‚Äã

Each provider has a corresponding default model, and users can also customize it.

ProviderDefault ValueOptions
Ignore--
Geminigemini-2.0-flashgemini-2.5-pro-preview-03-25, gemini-1.5-pro, gemini-1.5-flash, more
DashScopeqwen-turboqwen-max, qwq-plus, more

What is the purpose of the Ignore option?

  • Through debugging, we know the returned translation results, but there are always some translations that need manual adjustment. At this point, we can individually modify the text_trans value and save it.

  • In subsequent executions, if we want to use our manually adjusted translation, we can select the Ignore option.

Randomness ‚Äã

Controls the randomness of text generation. Lower temperature is more accurate, while higher temperature is more creative.

Diversity ‚Äã

Controls the depth of exploration during generation. Larger values usually generate more accurate text.

Batch Processing ‚Äã

Controls the length of returned content, processing long texts in batches.

When enabled, it can save token usage by prioritizing reading from the cache.
