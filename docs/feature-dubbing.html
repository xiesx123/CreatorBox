How to Choose‚ùì

CTTS requires high-performance devices, and synthesis efficiency is low on low-performance devices. It is recommended for regular users to use ETTS, which is faster and can be used to adjust other module parameters before switching to CTTS for dubbing. For users with high-performance devices or those looking to optimize dubbing quality, CTTS is recommended. Additionally, you can use Google Colab for remote deployment to improve processing speed.

WARNING

During the speech synthesis process, the speech rate value will be printed. This value should be close to 1 and should ideally not exceed 1.2, as it may cause pitch distortion. If unavoidable, manual adjustments can be made.

Preview ‚Äã

Users can input custom text for speech synthesis preview to confirm the dubbing effect. Switch between Basic -> Language to change the preview text language.

Dubbing List ‚Äã

Displays the currently available voice options, supporting preview, deletion, and reordering.

Operation Instructions:

  • Click Preview to immediately play the voice clip.
  • Click Delete to remove the voice.
  • Use Move Up / Move Down to control the order of voices.

Note

Example: When a user uploads test.mp4 and marks it as 001:

    1. For single-person dubbing, the Speaker defaults to 0.
    1. For multi-person dubbing, the Speaker should match the spk in test_001.json data.
