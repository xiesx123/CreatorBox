Support exporting materials to CapCut, providing fine-grained control over visuals, audio, subtitles, and more to assist with post-production adjustments and customization.

Configuration Options â€‹

The exported draft can be adjusted in CapCut.

Preview â€‹

  • Disabling preview will merge video materials locally, suitable for users without CapCut installed.

  • Enabling preview reduces merging wait time, allowing professional editing after exporting the draft (Recommended).

Draft Path â€‹

Open CapCut, click the bottom-right corner to view the Draft Location.

Copy the address and paste it.

Audio â€‹

  • Original Volume: The volume of the original video.

  • Speech Volume: The volume of the dubbed voice.

  • Background Volume: The volume of the original video after removing vocals.

Adjust as needed, with a range of [0,1].

Subtitles â€‹

  • Dubbed Subtitles: Subtitles generated from dubbing.

  • Original Subtitles: Subtitles from the original video.

Adjustable options include: Font Size, Font Height, Font Border Color, Font Color.

How to Set Values â“

Taking a 480*852 vertical screen as an example, horizontal screens are also applicable.

1. Size â€‹

Refer to CapCut -> Text -> Font Size.

2. Position â€‹

Refer to CapCut -> Text -> Position.

3. Color â€‹

Select directly from the color palette.

Border Color: #000000
