Error when installing Cython pynini ​

Solution
bash
conda install -c conda-forge Cython==3.0.12
conda install -c conda-forge pynini==2.1.5

or

bash
conda config --add channels conda-forge
conda install Cython==3.0.12
conda install pynini==2.1.5

Usage ​

Startup Error ​

Solution
  • If you see RuntimeError: this license key is expired, try forcing an update:
bash
python cli.py update --force
  • If you see RuntimeError: [WinError 10060]...(1:11076), Please try restarting:
bash
python cli.py start
  • On Windows systems, simply double-click start.bat to run.

interface Language ​

  • http://127.0.0.1:8000?lang=en English
