  • Optional

2. Network Proxy ‚Äã

  • Default
bash
python cli.py proxy -h 127.0.0.1 -p 10808
  • With authentication
bash
python cli.py proxy -h 127.0.0.1 -p 10810 -u admin -pwd 123456
  • For a specific site
bash
python cli.py proxy -h 127.0.0.1 -p 10810 -u admin -pwd 123456 -s https://www.youtube.com -t 5

Expected output:

log
üåê Starting proxy verify. -> https://www.youtube.com
‚úÖ Proxy verified successfully.

Tip

  • Once verified, no need to enable v2rayN global mode.
  • Any command-line tool that supports the --proxy option can use this parameter.
