VoxCPM1.5 ​

字数
251 字
阅读
2 分钟

VoxCPM1.5 领先的免分词 TTS，实现高表现力、上下文感知语音生成。支持中英双语，高效实现零样本声音克隆。

主界面 ​

`

启动方式 ​

  • 链接启动
http://127.0.0.1:8000/launcher?project=voxcpm
  • 命令启动
bash
uv run cli.py install -n voxcpm -p 8016 --start
  • 原生启动
bash
uv run app.py
