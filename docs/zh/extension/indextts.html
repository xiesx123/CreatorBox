IndexTTS2 ​

字数
468 字
阅读
3 分钟

IndexTTS2 是一个强大的自回归零样本文本转语音系统，实现了对语音生成前所未有的控制。具备高级情感表达、精确时长控制和即时声音克隆能力。

主界面 ​

启动方式 ​

  • 链接启动
http://127.0.0.1:8000/launcher?project=index_tts
  • 命令启动
bash
uv run cli.py install -n index_tts -p 8004 --start
  • 原生启动
bash
uv run webui.py --host 0.0.0.0 --port 8004 --fp16 --deepspeed

注意

  • 启动成功后，lang 需保持与页面语言一致，仅支持 zh_CN 或 en_US

  • 在配音模块中填写本地与远程地址，可以批量语音合成，例：

    • 本地访问:http://127.0.0.1:8004
    • 远程访问 : https://xxx.gradio.live、https://xxx.ngrok-free.app、https://xxx.loca.lt
    • 属于你的公网 IP 或域名
