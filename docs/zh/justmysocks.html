Just My Socks 🌐 ​

字数
3076 字
阅读

Just My Socks（JMS） 是一家知名高速线路上网服务提供商，由 搬瓦工 官方推出，专为海外上网提供稳定高效的解决方案。其服务支持系统自动切换 IP，有效避免因 IP 被封导致无法使用的情况，保障网络的速度与稳定性

此外，它还具备以下优势：

  • 搬瓦工官方出品，安全可靠 ：由搬瓦工官方运营，可信度高
  • 长期可用，稳定性绝佳 ：服务提供商保证长期可用性，并提供出色的稳定性
  • 支持 7 天无理由退款 ：如果在 7 天内对服务不满意，可以随时申请退款
  • 支持支付宝及银联卡付款 ：提供多种方便的付款方式，包括支付宝和银联卡
  • 高速 CN2 GIA 线路 ：使用高速的 CN2 GIA 线路，提供快速的网络连接
  • 被墙自动换 IP，无需担心 IP 被封 ：系统会自动切换节点，确保用户在 IP 被封锁的情况下仍然能够使用服务

官网 ​

由于某些原因，为方便用户访问官网，JMS 提供了多个镜像地址。用户可根据自身的网络情况选择以下任一地址进行访问、注册与购买操作

  • https://justmysocks.net （已封锁）
  • https://justmysocks2.net（中国大陆可以访问）
  • https://justmysocks3.net（中国大陆可以访问）
  • https://justmysocks6.net（中国大陆可以访问）

注意

所有的镜像地址之间数据是互通的

个人用户

  • 如果是个人使用，追求高性价比

    推荐选择 LA 500 套餐，提供了出色的性能和价格平衡，并支持同时连接 5 台设备。每个月 500G 的流量以及 2.5G 的带宽足够个人在 Youtube 上追剧、刷视屏等需求

  • 对于多用户多设备同时使用的情况

    LA 1000 套餐是一个不错的选择，它不限制设备数量，而且每个月 1000G 的流量完全可以支持起 5 个人以上的追剧、上网

  • 如果是个人对速度和延迟有较高要求，且没有大流量的需求的话

    建议您选择 Tokyo 100 或 Hong Kong 100。这些节点由于地理距离的优势，通常能提供较好的连接性能

企业用户

  • 对于企业用户，一般同时对速度、延迟、带宽和使用设备都有较高要求

    推荐考虑使用 Hong Kong 500 或 Hong Kong 1000 套餐。这些套餐提供了速度和稳定性的保证，适合满足企业级需求

在完成账号激活后，我们就可以登录后台查看节点的配置信息了。在菜单中选择 Services > My Services，就可以看到你购买的产品信息，列表中的 状态 显示可用，即为有效节点

点击这个可用的套餐进去就可以看到套餐的节点配置信息了

配置节点 ​

JMS 支持如下五种方式进行配置

配置方式

  • 订阅地址方式配置
  • 链接或二维码方式配置
  • IP或二维码方式配置
  • API 配置
  • 手动配置

推荐使用 订阅地址方式配置节点 、链接或二维码方式配置节点

订阅方式配置 ​

使用 JMS，您可以轻松地使用订阅地址来自动化添加节点配置信息。只需复制订阅链接地址，即可在相关客户端自动配置，省去了手动配置的烦恼

在后台节点信息中，找到 Subscription 选项，下方找到一串以 https 开头的网址

点击 Copy 按钮即可，然后在客户端粘贴此订阅链接即可完成配置 S

链接或二维码方式配置 ​

一旦购买完套餐节点，系统将自动生成相应的二维码信息。如果客户端支持扫描二维码导入配置信息，可以采用这种便捷的方式进行节点配置

在后台节点信息中，找到 DNS Link + QR 选项，点击进入该选项，您将看到相关的链接和二维码

点击 Copy 就可以复制节点链接 或者 通过客户端扫描对应二维码或者，然后在相关客户端软件导入链接即可

客户端 ​

Just My Socks（JMS） 是一个代理服务提供商，本身不提供官方客户端。用户可通过第三方的 Shadowsocks 或 V2Ray 客户端连接和使用其服务

Windows

支持多种代理协议，包括 VMess、VLESS、Trojan、Shadowsocks 等，可选择直连、智能分流或全局代理方式，适合日常浏览、访问受限网站或加密通信等场景

  • v2rayN 下载 - 功能强大、使用广泛的 V2Ray 图形客户端
  • Clash For Windows 下载 - 基于 Clash 核心，支持规则分流，适合高级用户

MacOS

为 macOS 用户打造的图形代理客户端，支持多种协议（如 VMess、Trojan、Shadowsocks 等），可配置规则分流，界面简洁易用，适合需要灵活控制流量的用户

  • v2rayU 下载 - 轻量且易于上手的 V2Ray 客户端
  • ClashX 下载 - 基于 Clash 的高级代理客户端，支持多协议和 YAML 配置文件

Android

安卓平台下主流的代理客户端应用，支持 VMess、Shadowsocks、Trojan 等协议，可导入配置文件或二维码，适用于移动端科学上网

  • v2rayNG 下载 - 基于 V2Ray 核心，操作简单、配置灵活
  • Clash for Android 下载 - 支持规则分流和多协议，适合进阶用户

iOS

功能强大的 iOS 代理客户端，支持 SS、SSR、V2Ray、Trojan 等协议，界面简洁、支持分应用代理。需使用外区 Apple ID 购买，参考价格 $2.99

  • Shadowrocket 下载 - iPhone/iPad 用户首选，支持导入多种配置和订阅
