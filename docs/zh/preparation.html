环境变量 ​

请根据你的需求，替换环境变量和相应的 API 密钥，所有信息均存储在本地电脑的环境变量中，程序不会记录保存 apikey 信息；例（OPENAI_API_KEY）

  1. 右键点击 此电脑 并选择 属性

  2. 点击 高级系统设置，然后选择 环境变量

  3. 在 系统变量 部分，点击 新建 并分别添加以下变量 OPENAI_API_KEY

  4. 进入命令提示符或PowerShell，运行以下命令验证：

powershell
echo $env:GOOGLE_API_KEY
cmd
echo %GOOGLE_API_KEY%

提示

Colab 使用，见 《远程部署》 章节

依赖软件 ​

推荐下载 集成安装包，内置便携集成环境，无需额外配置 🔥🔥🔥🔥🔥

Python 是一种高级编程语言，语法简洁、易读，适合快速开发、数据分析和自动化脚本。

  1. 下载安装
前往官网根据操作系统下载 Python 安装包
  1. 验证安装
bash
python --version

正常安装返回结果

log
Python 3.11.11

推荐下载 ​

Miniconda 是一个可以快速创建干净、隔离的 Python 虚拟环境，适合数据分析、AI 项目、自定义脚本等场景

  1. 下载安装

推荐使用 Miniconda 而非 Anaconda，体积更小、启动更快、依赖更纯粹

前往官网根据操作系统下载 Miniconda 安装包；这里以 `Windows` 为例

- 双击 `.exe` 安装包
- 勾选 “**Add Miniconda to PATH**”（可选但推荐）
- 完成后在开始菜单搜索 "Anaconda Prompt"
  1. 验证安装
bash
conda env list

正常安装返回结果

log
PS C:\Users\home> conda env list
# conda environments:
#
base                   D:\Projects\miniconda3
  1. 常见命令
命令说明
conda create --name creatorbox python=3.11.11创建名为 creatorbox 的环境，指定 Py 版本 3.11.11
conda env list列出所有已创建的虚拟环境
conda activate creatorbox激活进入名为 creatorbox 的环境
conda deactivate退出当前激活的环境
conda remove -n creatorbox --all删除名为 creatorbox 的整个环境
