环境变量 ​

替换环境变量和相应的 API 密钥，所有信息均存储在本地电脑的环境变量中，程序不会记录保存 apikey 信息；例（GEMINI_API_KEY）

  1. 右键点击 此电脑 并选择 属性

  2. 点击 高级系统设置，然后选择 环境变量

  3. 在 系统变量 部分，点击 新建 并分别添加以下变量 GEMINI_API_KEY

  4. 进入命令提示符或PowerShell，运行以下命令验证：

powershell
echo $env:GEMINI_API_KEY
cmd
echo %GEMINI_API_KEY%

提示

Colab 使用，见 《远程部署》 章节

依赖软件 ​

推荐下载 集成安装包，内置便携集成环境，无需额外配置 🔥🔥🔥🔥🔥

Aria2 是一个支持多线程下载的轻量级命令行工具。它可以将文件分成多个部分，并通过多个连接同时下载，大幅提高下载速度。

  1. 下载安装
前往官网根据操作系统下载 Aria2 安装包
  1. 验证安装
bash
aria2c --version

正常安装返回结果

log
aria2 version 1.37.0
