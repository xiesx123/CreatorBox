安装依赖报错 ​

解决方法
bash
# cython
conda install -c conda-forge Cython==3.0.12
# cosyvoice
# pynini
conda install -c conda-forge pynini==2.1.5

使用 ​

启动报错 ​

使用方式
  • 启动报错 RuntimeError: this license key is expired，请尝试强制更新
bash
python cli.py update --force
  • 启动报错 RuntimeError: [WinError 10060]...(1:11076)，请尝试重新启动
bash
python cli.py start
  • Windows 直接双击 start.bat 运行

界面语言切换 ​

目前仅支持中英，添加 ?lang= 参数，默认使用浏览器语言首选项，逐步适配中...

使用方式
  • http://127.0.0.1:8000?lang=zh 中文
  • http://127.0.0.1:8000?lang=en English

配音语言配置 ​

默认内置10种语言，支持自定义添加节点

使用方式
webapp\data\settings.json
json
{
    "locales": [
        {
            "name": "中国",
            "locale": "zh-CN",
            "country": "China",
            "enable": true
        }
    ]
    // ...
}
