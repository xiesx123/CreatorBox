Just My Socks (JMS) is a well-known high-speed internet service provider launched officially by BandwagonHost. It offers a stable and efficient solution for overseas internet access. The service supports automatic IP switching, effectively avoiding issues caused by IP bans and ensuring network speed and stability.

Additionally, it has the following advantages:

  • Officially operated by BandwagonHost, safe and reliable: Managed by BandwagonHost, ensuring high credibility.
  • Long-term usability with excellent stability: The provider guarantees long-term usability and outstanding stability.
  • 7-day no-questions-asked refund policy: If dissatisfied within 7 days, you can request a refund anytime.
  • Supports Alipay and UnionPay payments: Offers multiple convenient payment methods, including Alipay and UnionPay.
  • High-speed CN2 GIA lines: Utilizes high-speed CN2 GIA lines for fast network connections.
  • Automatic IP switching to avoid bans: The system automatically switches nodes to ensure uninterrupted service even if an IP is banned.

Subscription Configuration ‚Äã

With JMS, you can easily add node configuration using a subscription URL. Simply copy the subscription link and paste it into the client to automate the setup.

In the dashboard, locate the Subscription section and copy the URL starting with https.

Paste this link into your client to complete the configuration.

Link or QR Code Configuration ‚Äã

After purchasing a plan, the system generates a corresponding QR code. If your client supports QR code scanning, you can use this method for quick setup.

In the dashboard, locate the DNS Link + QR section. Click to view the link and QR code.

Copy the link or scan the QR code with your client to import the configuration.

Just My Socks (JMS) is a proxy service provider and does not offer official clients. Users can connect using third-party clients like Shadowsocks or V2Ray.

Windows

Supports multiple proxy protocols, including VMess, VLESS, Trojan, and Shadowsocks. Suitable for browsing, accessing restricted websites, or encrypted communication.

  • v2rayN Download - A powerful and widely used V2Ray GUI client.
  • Clash For Windows Download - Based on the Clash core, supports rule-based routing, ideal for advanced users.

MacOS

Designed for macOS users, supporting multiple protocols (e.g., VMess, Trojan, Shadowsocks). Features rule-based routing and a user-friendly interface.

  • v2rayU Download - A lightweight and easy-to-use V2Ray client.
  • ClashX Download - An advanced proxy client based on Clash, supporting multiple protocols and YAML configurations.

Android

Popular proxy client apps for Android, supporting VMess, Shadowsocks, Trojan, and more. Allows importing configurations via files or QR codes.

  • v2rayNG Download - Based on the V2Ray core, simple and flexible.
  • Clash for Android Download - Supports rule-based routing and multiple protocols, ideal for advanced users.

iOS

Powerful iOS proxy clients supporting SS, SSR, V2Ray, Trojan, and more. Features a clean interface and app-specific proxy settings. Requires an overseas Apple ID for purchase, priced at $2.99.

  • Shadowrocket Download - A top choice for iPhone/iPad users, supporting multiple configurations and subscriptions.
