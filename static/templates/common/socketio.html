CreatorBox Socket API Documentation
  • ▾OBS6
    • EVENTget_obs_status获取 OBS 连接状态
    • EVENTconnect_obs连接到 OBS
    • EVENTdisconnect_obs断开 OBS 连接
    • EVENTget_streaming_status获取直播状态
    • EVENTstart_streaming启动直播
    • EVENTstop_streaming停止直播
  • ▾Scene7
    • EVENTget_current_scene获取当前场景
    • EVENTget_scenes获取所有场景
    • EVENTswitch_scene切换场景
    • EVENTget_scene_sources获取指定场景的采集源
    • EVENTget_source_info获取采集源信息
    • EVENTcontrol_source控制采集源播放/暂停/停止/显隐
    • EVENTset_source_file设置采集源视频文件路径
  • ▾dh8
    • EVENTget_dh_camera_list获取摄像头列表
    • EVENTget_dh_status获取 DH 运行状态
    • EVENTstart_dh_streaming启动 DH 数字人流程
    • EVENTstop_dh_streaming停止 DH 数字人流程
    • EVENTinterrupt_dh_audio中断当前音频播放
    • EVENTclear_dh_audio清空音频队列
    • EVENTsend_dh_microphone发送麦克风录音指令
    • EVENTsend_dh_audio发送音频文件到播放队列
  • ▾Dubb3
    • EVENTget_dubb_names获取配音项目列表
    • EVENTget_dubb_suffixes获取配音项目的后缀列表
    • EVENTget_dubb_info获取配音信息（音频列表+视频路径）
  • ▾Room3
    • EVENTconnect_room连接抖音直播间
    • EVENTdisconnect_room断开直播间连接
    • EVENTgenerate_reply生成直播间回复文本
